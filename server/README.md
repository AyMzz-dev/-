# 小七授权服务端 - 易语言版

易语言编写的云端授权验证服务端，配合小七授权系统 PHP 客户端使用。

## 功能特性

- **HTTP 服务端**监听指定端口，接收 PHP 客户端请求
- **SQLite 数据库**存储授权信息，自动创建数据表（单文件 `auth.db`，无需安装 MySQL）
- **授权管理 GUI** 图形界面添加/启用/禁用/删除授权
- **支持两种请求**：`checkauth` 验证授权、`sendmail` 发送邮件
- **访问日志**所有请求记录数据库，GUI 实时日志输出
- **一键生成密钥**随机生成 64 位密钥，适合复制给客户

## 编译环境

需要以下模块（易语言官方支持库+第三方模块）：

| 名称 | 说明 |
|------|------|
| `网络服务器` | 系统核心支持库自带 |
| `Sqlite数据库` | 易语言自带支持库 |
| `JSON 类` | 内置的 `class_json` |
| `超级列表框` | 系统核心支持库自带 |

## 使用方法

### 1. 数据库

**无需额外配置！** SQLite 数据库文件 `auth.db` 会在程序同目录自动创建，首次启动即可使用。

### 2. 配置参数

打开服务端 GUI：
- 数据库路径默认为程序同目录下的 `auth.db`，无需修改
- 服务端口默认 `99`，可以自行修改

### 3. 启动服务

点击**启动服务**，等待：
```
[数据库] 连接成功 127.0.0.1:3306
[数据库] 数据表初始化完成
[服务器] 启动成功 HTTP://0.0.0.0:99
```

### 4. 添加授权

在 GUI 下方：
1. 填写**域名**（客户域名，如 `example.com`）
2. 点击**生成密钥**（自动生成 64 位随机密钥）
3. 填写**备注**（可选，比如客户名称）
4. 点击**添加授权**

### 5. 在 PHP 客户端配置

PHP 客户端在 `function/function_core.php` 修改服务端地址：

```php
$ServerURL = 'http://你的IP:端口/api.php';
```

**注意**：PHP 客户端会自动尝试多个备用服务器地址，第一个失败自动试第二个，以此类推。

### 6. 管理授权

在授权列表右键点击：
- **启用** - 启用已禁用的授权
- **禁用** - 禁用后客户端无法使用
- **删除** - 删除授权（不可恢复）

## API 协议

### `checkauth` 验证授权

请求：
```
GET /api.php?mod=checkauth&domain=客户域名&sitekey=站点密钥
```

返回 JSON：

| code | 含义 |
|------|------|
| `1` | 授权有效，正常运行 |
| `0` | 未找到授权/参数错误 |
| `-3` | 授权已被禁用 |

```json
{"code": 1, "msg": "授权有效"}
```
```json
{"code": -3, "msg": "您的授权已被封禁", "tips": "您的授权已被封禁，请联系管理员QQ:xxxxxx"}
```

### `sendmail` 发送邮件

请求：
```
POST /api.php?mod=sendmail&domain=客户域名&sitekey=站点密钥
Content-Type: application/x-www-form-urlencoded

title=邮件标题&content=邮件内容&tomail=收件人邮箱
```

返回：
```json
{"code": 1, "msg": "发送成功"}
```

## 客户端回发信息

PHP 客户端每次验证会自动回发：
- 域名
- 站点密钥
- 客户端 IP
- 请求结果

全部记录在 `auth_logs` 表中，可以查看。

## 防火墙开放端口

记得在服务器防火墙开放你配置的端口（默认 `99`）：

```bash
# CentOS 7+ firewalld
firewall-cmd --permanent --add-port=99/tcp
firewall-cmd --reload
```

## 安全建议

1. 建议服务端和客户端不在同一台服务器，客户端被攻破不影响服务端
2. 定期备份数据库
3. 密钥长度使用默认 64 位，不要太短
4. 保持服务端程序更新

## 协议
